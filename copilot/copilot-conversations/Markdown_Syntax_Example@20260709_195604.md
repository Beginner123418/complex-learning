### 一、基本语法
#### 1. 标题（Headings）
- 使用 `#` 到 `######` 表示 1 至 6 级标题，如：
  ```markdown
  # 一级标题
  ## 二级标题
  ```
- 可选语法：在文本下方用 `==` 或 `--` 标识标题（不推荐）。

#### 2. 段落（Paragraphs）
- 空行分隔段落，避免缩进：
  ```markdown
  这是第一段。

  这是第二段。
  ```

#### 3. 换行（Line Breaks）
- 行尾添加 **两个或多个空格** 或使用 `<br>` 标签：
  ```markdown
  第一行。  
  第二行。
  ```
  ```markdown
  第一行。<br>
  第二行。
  ```

#### 4. 强调（Emphasis）
- **粗体**：`**文本**` 或 `__文本__`
- *斜体*：`*文本*` 或 `_文本_`
- 粗+斜体：`***文本***` 或 `___文本___`
- **注意**：单词中间使用星号（如 `*非首尾*`）以确保兼容性。

#### 5. 列表（Lists）
- **有序列表**：数字 + 句点（`.`），如：
  ```markdown
  1. 第一项
  2. 第二项
  ```
- **无序列表**：使用 `-`、`*` 或 `+`，统一风格：
  ```markdown
  - 列表项 A
  - 列表项 B
  ```
- **嵌套列表**：缩进子项：
  ```markdown
  1. 主项
      - 子项
  ```

#### 6. 链接（Links）
- 格式：`[显示名](<URL> "标题")`
- 示例：`[链接](https://example.com "说明")`
- **网址/邮箱**：直接使用尖括号包裹：
  ```markdown
  <https://example.com>
  <email@example.com>
  ```

#### 7. 图片（Images）
- 格式：`![替代文本](<图片链接> "标题")`
- 链接图片：`[![替代文本](图片链接)](跳转链接)`

#### 8. 代码（Code）
- **行内代码**：反引号包裹 `code`
- **代码块**：缩进 4 空格或使用围栏式语法：
  ```markdown
  ```
  code block
  ```
  ```

#### 9. 引用（Blockquotes）
- 使用 `>` 标记引用内容，多段落需每行加 `>`：
  ```markdown
  > 第一段。
  >
  > 第二段。
  ```

#### 10. 分隔线（Horizontal Rules）
- 使用 `---`、`***` 或 `___` 单独成行，并前后保留空行。

### 二、扩展语法
#### 1. 表格（Tables）
- 标题行用 `|---|` 分隔列，内容行用 `|内容|`：
  ```markdown
  | 列1 | 列2 |
  | --- | --- |
  | 内容A | 内容B |
  ```
- **对齐方式**：在标题行连字符两侧添加冒号（如 `:---` 左对齐）。

#### 2. 围栏代码块（Fenced Code Blocks）
- 使用三个反引号包裹，指定语言实现语法高亮：
  ```markdown
  ```python
  print("Hello, world!")
  ```
  ```

#### 3. 脚注（Footnotes）
- 引用：`[^1]`
- 定义：`[^1]: 注释内容`
- 示例：
  ```markdown
  这是脚注示例 [^1].
  ```

#### 4. 其他扩展
- **自动链接**：URL 和邮箱自动转为可点击链接。
- **任务列表**：`[ ]` 表示未完成，`[x]` 表示完成：
  ```markdown
  - [ ] 待办事项
  - [x] 已完成
  ```
- **删除线**：`~~被删除的文字~~`

---

### 三、最佳实践
| 类别         | 建议                           |
|------------|-------------------------------|
| 标题        | `#` 与文字间保留一个空格           |
| 段落       | 不要缩进                        |
| 换行       | 使用两个空格或 `<br>`             |
| 强调       | 单词中间用星号（如 `*非首尾*`）    |
| 列表       | 分隔符统一，数字后仅用句点 `.`     |
| 链接       | URL 中的空格替换为 `%20`          |
| 分隔线      | 前后保留空白行                   |
