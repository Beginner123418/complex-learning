# [不沾一滴酒的回答](https://www.zhihu.com/question/2052670200672212964/answer/2058214788988195451)

1966年2月，老人家在与侄子毛远新的谈话中，提到了黄埔军校。老人家恰恰认为，黄埔军校最培养人才的时候就是半年制时期，后来学制拉长了，反而打不好仗：

> 理工科有它的特殊性，有它自己的语言，要读一点书。但是也有共性，光读书不行。**黄埔军校就读半年，毕业后当一年兵，出了不少人材**。改成陆军大学以后，结果出来尽打败仗，作我们的俘虏。

说白了师傅领进门，修行靠个人，黄埔名校给学生提供了打仗最重要的东西，一个志同道合、薪火相传的卓越平台，以及肝胆相照、守望相助的精英人脉网络，至于学生们自己能走多远，能消化掉多少资源，就看学生自己的悟性和能力了。

打仗打的就是人情世故，古今中外，莫不如是，拉不起山头连底下将领都指挥不动，那还打个der啊。

黄埔军校半年里，老师能教的都教了。学生自己想上进、要上进，边实践边读书，或者选择进一步进修，一个中高级军官能接触到的资源，总比一个军校生强得多。李诚毅回忆说当年200师为了部队训练参考，干脆自行成立了一个编译处工厂，向欧美各国去搜购原本书籍来从事译印，分发给各位军官。这是普通军校生能做得到的吗？

邱清泉在柏林陆军大学深造时，学校歧视外国学生，往往保留最新战术之讲义不发，邱清泉便“用种种方法，将其取得，连夜打字抄录，秘为保存”。我相信那些唾手可得的德国同学们，很多人纯粹为了混张名校文凭，拿个好绩点，对实战应用毫无兴致。

老人家没接受过正规军事教育，也不影响他总结经验、因地制宜，取得了一次次辉煌的胜利。最近的某片，台词里那个因为自己没留过学，就毫无自信、自低自卑的劲儿，别说放老人家身上，就是安给凯申，那都是在黑凯申：

# [兔子爷爷的回答](https://www.zhihu.com/question/2052670200672212964/answer/2053501785743087407)

能进黄埔的，无论是智力，体能还是毅力，在当是的中国至少也是前10%的存在

只要运气不错，没被阎王爷带走，历经千战百战自然打出来了

# [大相邦的回答](https://www.zhihu.com/question/2052670200672212964/answer/2053508909319578833)

自古以来所有的好学校都不是学校有多NB。而是能招一堆NB的学生。黄埔军校只是一座军校，但是它的价值观招来了一大批有志青年。他们满腔热血希望救国。黄埔军校只是给了他们一个机会罢了。

# [就这样凝视着你的回答](https://www.zhihu.com/question/2052670200672212964/answer/2053205156972564584)

确实是短训班，但时势造英雄，黄埔军校毕业的学生都是边打边学，活下来自然就能成长，活不下来就是他运气不好，战场实践是最好的老师。

# [右仆射同平章事的回答](https://www.zhihu.com/question/2052670200672212964/answer/2053912942454871189)

因为机会多，历练试错的机会多，

一个中等之资，只要不断有机会试错历练，也能成为上上之才。一将功成万骨枯，不断的大浪淘沙下来，活下来的机会越集中，带兵出战的机会越多，可不就成为名将了吗？

黄埔六期之后，虽然学制上来了，改为了四年长训制，但是位置已经被前面的学长占据了，自然出头的就少。大家熟悉的黄埔名将，大都是四期之前的。

同样道理的，还有恢复高考后的老三届，也成才率特别高，副部级以上一堆堆，也是这个舞原因啊。
