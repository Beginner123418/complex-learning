# 数学公式推导与解题：英语母语级用语全指南

> 适用领域：初等代数、线性代数、抽象代数、复分析（复变函数）  
> 核心目标：提供从“读题”到“收尾”的全流程地道英语表达，杜绝中式数学英语。

---

## 第一部分：逻辑连接与拉丁缩写（基础底盘）

在推导中，逻辑连接词决定句子的严谨性。以下是在数学写作中高频出现的缩写与全称。

| 缩写 | 拉丁/英文全称 | 中文含义 | 核心用法与致命误区 |
| :--- | :--- | :--- | :--- |
| **i.e.** | *id est* | 即、也就是说 | 用于**重新表述**前文。误区：不可用于举例。例：*A square matrix, i.e., a matrix with equal rows and columns.* |
| **e.g.** | *exempli gratia* | 例如 | 用于**举例**。误区：不要加 *etc.*，因为 e.g. 已暗示列举不完全。 |
| **s.t.** | *subject to / such that* | 满足、受约束于 | 优化中表约束（*subject to*），集合中表条件（*such that*）。例：*∃ x s.t. x>0*。 |
| **w.r.t.** | *with respect to* | 关于、相对于 | 求导或偏导时必用。例：*Differentiate w.r.t. x.* |
| **w.l.o.g. (WLOG)** | *without loss of generality* | 不失一般性 | 假设特殊情况。误区：若假设不显然，需简略说明为何不失一般性。 |
| **iff** | *if and only if* | 当且仅当 | 逻辑等价。用于定义或充要条件证明。 |
| **Q.E.D.** | *quod erat demonstrandum* | 证毕 | 置于证明末尾。现代数学常用方框 `□` 代替。 |
| **cf.** | *confer* | 参照、比较 | 引导读者参考其他文献或图表。 |
| **N.B.** | *nota bene* | 请注意 | 强调极其重要的细节，通常大写。 |
| **resp.** | *respectively* | 分别地 | 用于两组元素一一对应。例：*a and b equal 1 and 2, respectively.* |
| **viz.** | *videlicet* | 即、详细说明 | 比 i.e. 更正式，用于具体列举。现代论文已少见。 |
| **et al.** | *et alii* | 等人 | 仅用于参考文献中列举多位作者。 |

**注意事项**：
- 拉丁缩写（i.e., e.g., cf.）在科技写作中**不使用斜体**。
- i.e. 和 e.g. 后面通常跟逗号（美式风格），且每个字母后都有句点。
- *et al.* 中只有 *al.* 后有句点，因为 *et* 本身不是缩写。

## 第二部分：处理方法与代数运算（动作动词）

公式推导的核心在于“操作”。下表区分了极易混淆的近义词（尤其是代入）。

| 中文操作 | 地道英文动词 | 标准句式（介词绝对要记牢） | 母语者点评 |
| :--- | :--- | :--- | :--- |
| **代入（重点）** | *Substitute* | **Substitute A into B**（将A代入B式）<br>**Substitute A for B**（用A替换B） | 介词反则意思完全相反。最安全写法：*Plug A into B*（口语但通用）。 |
| **替换** | *Replace* | *Replace A with/by B*（用B换掉A） | 被动语态常用：*A is replaced by B.* |
| **化简** | *Simplify* | *Simplifying the L.H.S., we get...* | 通常放在句首作分词状语。 |
| **抵消/约去** | *Cancel / Cancel out* | *The terms \( x^2 \) cancel.* | **不及物动词**！千万别写 *are canceled*（除非指被人为划掉）。 |
| **展开（去括号）** | *Expand* | *Expanding the brackets yields...* | 若展开乘积，也可用 *Multiply out*。 |
| **因式分解** | *Factor / Factorize* | *Factorizing the quadratic...* | 美式常用 *Factor*，英式常用 *Factorise*。 |
| **提取公因式** | *Factor out* | *Factor out the common term \( e^x \).* | 介词 **out** 不可省略，否则变成“分解整个式子”。 |
| **配方** | *Complete the square* | *Completing the square gives \( (x-1)^2 = 4 \).* | 固定搭配，不可说 *make the square*。 |
| **分离变量** | *Isolate* | *Isolate \( y \) on the left-hand side.* | 强调“单独拎出来”。 |
| **消元（方程组）** | *Eliminate* | *Eliminate \( t \) from (1) and (2).* | 常与 *from* 搭配。 |
| **恒等于** | *Identically equal* | *The expression is identically zero.* | 符号用 `\(\equiv\)`。只说 *is zero* 意味着只在某点成立。 |
| **求导** | *Differentiate* | *Differentiate both sides w.r.t. \( x \).* | 若对隐函数求导，常用 *implicit differentiation*。 |
| **积分** | *Integrate* | *Integrate the equation term by term.* | *term by term*（逐项）是级数积分的标准用语。 |
| **估值/放缩** | *Bound / Estimate* | *We bound the remainder term by \( O(1/n) \).* | 用 *by* 连接放缩的上界。 |

## 第三部分：因果与解题起点（读题与引入）

“由题可知”在英语中有严格的场合区分，不可混用。

| 中文意图 | 地道英文用语 | 适用条件 | 例句 |
| :--- | :--- | :--- | :--- |
| **由题干文字可知** | *The problem states that...* | 题干中有明确的文字描述（如几何关系）。 | *The problem states that the sequence is bounded.* |
| **根据题设/假设** | *By the hypothesis / By assumption* | 题目明确给出的数学前提。 | *By hypothesis, f is continuous on [a,b].* |
| **给定条件给出** | *We are given that... / It is given that...* | 最正式、最常用。 | *We are given that the matrix is symmetric.* |
| **众所周知/既然** | *Since...*（黄金词汇） | 引出既定事实或上一步结果。 | *Since the discriminant is negative, there are no real roots.* |
| **鉴于/给定** | *Given that...* | 侧重“在……前提下”。 | *Given that \( f \) is entire, its Taylor series converges everywhere.* |
| **因为（强调原因）** | *Because...* | 解释人为选择某操作的理由，放句中更自然。 | *We choose this contour because it avoids the singularity.* |
| **由(1)式可知** | *From (1), it follows that...* | 交叉引用上一步推导结果。 | *From (1), it follows that the limit exists.* |
| **结合(1)和(2)** | *Combining (1) and (2) yields...* | 多式联立时使用。 | *Combining (1) and (2) yields the desired inequality.* |

**高频错误警示**：
- 写 *Since A, so B* 是中式英语，语法错误。应写 *Since A, B* 或 *A, so B*。
- *Due to* 是形容词性介词短语，只能修饰名词（如 *The error is due to rounding*），**不可**放句首。若要放句首，请用 *Owing to* 或 *Because of*，但数学推导中建议直接用 *Since* 或 *Because* 引导完整从句。

## 第四部分：代数领域专属用语（初等 + 线性 + 抽象）

### 4.1 初等代数（方程与多项式）

| 中文 | 地道英文用语 | 实战例句 |
| :--- | :--- | :--- |
| 解方程（求未知数） | *Solve for \( x \)* | *Solve the equation for \( x \).* |
| 方程的根 / 零点 | *Roots / Zeros* | *The roots of this polynomial are complex.* |
| 判别式 | *The discriminant* | *Since the discriminant is negative, there are no real roots.* |
| 增根 / 验根 | *Extraneous root / Check / Verify* | *We must check for extraneous roots introduced by squaring both sides.* |
| 分离系数（多项式恒等） | *Equate the coefficients* | *Equating the coefficients of \( x^2 \), we get \( a=1 \).* |
| 因式定理 | *Factor theorem* | *By the factor theorem, \( (x-1) \) divides \( f(x) \).* |

### 4.2 线性代数（矩阵与向量空间）

| 中文 | 地道英文用语 | 核心介词/语法陷阱 |
| :--- | :--- | :--- |
| 矩阵的转置 / 逆 / 行列式 | *Transpose / Inverse / Determinant* | 介词用 **of**：*the determinant of A*。 |
| 矩阵可逆 / 非奇异 | *Invertible / Nonsingular* | *A is invertible iff det(A) ≠ 0.* |
| 矩阵的秩 | *Rank* | *The rank of A is 2.* |
| 线性相关 / 无关 | *Linearly dependent / independent* | 副词修饰形容词，不可写 *linear independent*。 |
| 向量张成空间 | *Span*（动词/名词） | *These vectors span \( \mathbb{R}^3 \).* |
| 特征值 / 特征向量 | *Eigenvalue / Eigenvector* | 无连字符，复数形式：*eigenvalues*。 |
| 对角化 | *Diagonalize* | *We can diagonalize \( A \) by using the eigenbasis.* |
| 内积（点积） | *Inner product / Dot product* | *The dot product of \( u \) and \( v \) is zero.*（正交 *orthogonal*） |
| 行阶梯形矩阵 | *Row echelon form* | *Reduce the matrix to row echelon form.*（介词用 **to**） |
| 线性变换的作用 | *Apply \( A \) to \( v \) / Act on* | **严禁**说 *Calculate A times v*，太口语化且不专业。 |

### 4.3 抽象代数（群、环、域）

| 中文 | 地道英文用语 | 绝对要避免的错误 |
| :--- | :--- | :--- |
| 二元运算 / 封闭性 | *Binary operation / Closed under* | 介词固定用 **under**：*closed under addition*。 |
| 结合律 / 交换律 | *Associative / Commutative* | *The operation is associative.* |
| 单位元 / 逆元 | *Identity element / Inverse* | *Every element has an inverse.* |
| 群 / 环 / 域 | *Group / Ring / Field* | 直接用，不加冠词也可（*G is a group*）。 |
| 同态 / 同构 | *Homomorphism / Isomorphism* | *There is a natural isomorphism between G and H.* |
| 核与像 | *Kernel / Image* | *The kernel of the homomorphism is a normal subgroup.* |
| 商群 / 商环 | *Quotient group / Quotient ring* | *We define the quotient group \( G/N \).* |
| 由……生成 | *Generated by* | *The ideal generated by \( x \) and \( y \).* |
| 元素的阶 | *Order of an element* | *The order of \( g \) divides the order of the group.* |
| **良定义** | ***Well-defined*** | **绝对高频词**！在商结构或等价类上定义映射时，必须证明 *This map is well-defined.* |

## 第五部分：复变函数（复分析）专属用语

复分析是术语密度最高、介词搭配最敏感的领域。

### 5.1 解析性与奇点（Holomorphicity & Singularities）

| 中文                  | 地道英文用语                                        | 核心介词与禁忌                                                                                                      |
| :------------------ | :-------------------------------------------- | :----------------------------------------------------------------------------------------------------------- |
| 在点 \( z_0 \) 处解析/全纯 | *Holomorphic / Analytic **at** \( z_0 \)*     | 介词用 **at**（点）。若说 *in a domain* 表示在整个区域内。                                                                     |
| 在区域 \( D \) 内全纯     | *Holomorphic **on** \( D \) / **in** \( D \)* | 两者均可，但 **on** 更强调区域作为定义域。                                                                                    |
| 亚纯函数                | *Meromorphic function*                        | 指除极点外全纯。**严禁**说 *Almost holomorphic*。                                                                        |
| 整函数                 | *Entire function*                             | 指在整个复平面 \( \mathbb{C} \) 上全纯（如多项式、指数函数）。                                                                     |
| 共形映射（保角）            | *Conformal mapping*                           | 本质是导数非零的全纯单射。                                                                                                |
| 柯西-黎曼方程             | *Cauchy-Riemann equations*                    | 通常缩写为 **C-R equations**。                                                                                     |
| 可去奇点                | *Removable singularity*                       | *The singularity at \( z=0 \) is removable.*                                                                 |
| 极点                  | *Pole*                                        | *\( f \) has a **simple pole** at \( z=0 \).*（一阶）<br>*\( f \) has a **pole of order \( m \)** at \( z_0 \).* |
| 本性奇点                | *Essential singularity*                       | *By Casorati–Weierstrass, near an essential singularity, the function is dense.*                             |
| 留数                  | *Residue*                                     | **固定搭配**：*The residue **of** \( f \) **at** \( z_0 \) is...*（绝对不写 *in*）                                      |

### 5.2 围道积分与分支切割（Contour Integration & Branch Cuts）

| 中文 | 地道英文用语 | 实战例句 |
| :--- | :--- | :--- |
| 围道 / 闭合路径 | *Contour / Closed curve* | *Consider the contour \( C \) consisting of the real interval and the upper semicircle.* |
| 正方向（逆时针） | *Positively oriented* | *Let \( C \) be the positively oriented unit circle.* |
| 缠绕 / 围绕 | *Wind around / Encircle* | *The contour winds around the origin once.* |
| 变形围道 | *Deform the contour* | *By Cauchy's theorem, we can deform the contour without changing the integral.* |
| 令半径趋于无穷 | *Let \( R \to \infty \)* | *Taking the limit as \( R \to \infty \), the integral over the large arc vanishes.* |
| 大圆弧 / 上半圆 | *Large arc / Semicircle* | *The integral over the semicircular arc vanishes by the ML inequality.* |
| 积分主值 | *Principal value (P.V.)* | *We take the Cauchy principal value of the integral.*（处理实轴奇点） |
| 多值函数 | *Multi-valued function*（或称 *relation*） | 母语者不喜欢叫 *function*，常说 *multi-valued relation*。 |
| 分支切割 | *Branch cut* | *Choose a branch cut along the negative real axis.* |
| 分支点 | *Branch point* | *\( z=0 \) and \( z=\infty \) are the branch points of \( \log z \).* |
| 单值分支 | *Single-valued branch* | *We select the principal branch of the logarithm.* |
| 辐角 | *Argument (arg)* | *Fix the argument of \( z \) such that \( 0 < \arg z < 2\pi \).* |

### 5.3 五大核心定理的标准表述

| 定理中文 | 英文标准表述 | 推导中的惯用承接语 |
| :--- | :--- | :--- |
| 柯西积分定理 | *Cauchy's Integral Theorem* | *By Cauchy's theorem, the integral along the closed contour vanishes.* |
| 柯西积分公式 | *Cauchy's Integral Formula* | *Applying Cauchy's integral formula to \( f \) at \( z_0 \), we get...* |
| 留数定理 | *Residue Theorem* | *By the Residue Theorem, the contour integral equals \( 2\pi i \) times the sum of the enclosed residues.*（**必须**用 *enclosed* 形容奇点） |
| 最大模原理 | *Maximum Modulus Principle* | *The maximum modulus principle implies that \( |f| \) attains its maximum on the boundary.* |
| 辐角原理 | *Argument Principle* | *The argument principle relates the change in argument to zeros minus poles.* |
| 儒歇定理 | *Rouché's Theorem* | *By Rouché's theorem, \( f \) and \( g \) have the same number of zeros inside the contour.* |

## 第六部分：介词陷阱速查表（高频易错）

| 中文意图 | 错误介词（×） | 正确介词（√） | 原因/记忆技巧 |
| :--- | :--- | :--- | :--- |
| 代入 A 到 B 中 | *Substitute A **to** B* | *Substitute A **into** B* | *into* 表示“进入”B式内部。 |
| 用 A 替换 B | *Replace A **by** B*（易混淆） | *Replace A **with** B* | 想“换掉”谁，谁就是第一个词。 |
| 留数在某点 | *Residue **in** z0* | *Residue **at** z0* | 点状位置用 *at*。 |
| 在围道内部 | *In the contour*（表示在线上） | *Inside the contour*（表示内部） | *In* 表示在围道这条线上，*Inside* 表示被包围的区域。 |
| 关于变量求导 | *Differentiate **by** x* | *Differentiate **w.r.t.** x* | 专业缩写 *w.r.t.* = with respect to。 |
| 在区域上全纯 | *Holomorphic **at** D* | *Holomorphic **on/in** D* | 区域用 *on/in*，点用 *at*。 |
| 封闭性（代数） | *Closed **to** addition* | *Closed **under** addition* | 固定搭配，*under* 表示“在……运算下”。 |
| 由(1)式可知 | *From (1) **we know***（冗长） | *From (1), **it follows*** | *Follows* 是数学推导的标准承接连词。 |

## 第八部分：推导常用“承接词”速记清单

在书写推导过程时，以下词语能让步骤行云流水：

| 动作阶段 | 首选动词/短语 | 备选词 |
| :--- | :--- | :--- |
| 开始推导 | *We begin by...* | *Let us first...* |
| 得出结果 | *Yields / Gives* | *Leads to / Produces* |
| 因此 | *Thus / Hence / Therefore* | *Consequently / So* |
| 注意 | *Note that...* | *Observe that...* |
| 另一方面 | *On the other hand...* | *Conversely...* |
| 综上所述 | *In conclusion / Summarizing* | *To sum up* |
| 这与……矛盾 | *This contradicts...* | *...which is absurd.* |
